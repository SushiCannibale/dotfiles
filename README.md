### Bindings

| Bind            | Command                                  |
|-----------------|------------------------------------------|
| CTRL+SPACE      | accept autosuggets                       |
| CTRL+->        | move one word forwards                   |
| CTRL+<-        | move one word backwards                  |
| SUPPR           | forward deletes a character              |
| CTRL+SUPPR      | forward deletes a word                   |
| CTRL+BACKSPACE  | backward deletes a word                  |
